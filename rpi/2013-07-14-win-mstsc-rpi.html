# windows远程连接树莓派

======

我不想用VNC，想用WINDOWS的远程桌面连接树莓派，如何实现？

需要在raspbian下面安装一个服务xdrp,输入如下命令

sudo apt-get install xrdp

然后就可以用win的远程桌面mstsc连接了
