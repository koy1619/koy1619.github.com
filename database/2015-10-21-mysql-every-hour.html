mysql取每隔一小时数据

chris 2015/10/21 mysql sql 
SELECT 
    HOUR(`input_date`) AS HOUR,
    COUNT(1) AS 交易笔数,
    brand_name_cn AS 品牌,
    DATE_FORMAT(`input_date`, '%m%d') AS 交易日期 
FROM
    test.`1023` 
WHERE input_date >= 20150930000000 
    AND input_date <= 20150930235959 
    AND brand_name_cn = '统一星巴克' 
GROUP BY HOUR ;
