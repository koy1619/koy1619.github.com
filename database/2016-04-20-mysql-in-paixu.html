MySQL查询按IN的顺序输出结果

chris 2016/4/20 mysql sql

在用 SELECT 查询的时候，如果用到了 IN ，那么查询结果中的顺序并不是按照 IN 后面所给的顺序返回，而是按照默认的升序排列。

而如果想要让查询结果按照 IN 里面给的顺序的话，有最简单方法：

SELECT 
    * 
FROM
    a 
ORDER BY FIELD(
        id,
        '16556004',
        '16552005',
        '16558009',
        '16551033'
    )
